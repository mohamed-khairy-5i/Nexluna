**Nexluna** أداة ويب عربية مجانية وسريعة لتحويل وحدات القياس بدقة عالية: الطول، الوزن، المساحة، درجة الحرارة، والحجم والسعة — على جميع الأجهزة، وبدون اتصال بالإنترنت بعد أول زيارة.

🔗 **الموقع المباشر:** <https://nexluna.netlify.app/>

---

## ✨ الميزات

- **خمس فئات تحويل:** الطول، الوزن، المساحة، درجة الحرارة، والحجم — بمعاملات رسمية دقيقة.
- **نتائج فورية** أثناء الكتابة، بدقة تصل إلى ست خانات عشرية.
- **دعم عربي كامل (RTL)** مع خط Cairo وتصميم متوافق مع الاتجاه من اليمين إلى اليسار.
- **تطبيق ويب تقدّمي (PWA):** قابل للتثبيت ويعمل دون اتصال بعد أول زيارة.
- **تحسين محركات البحث (SEO):** عناوين ووصف فريد لكل صفحة، خريطة موقع، بيانات منظّمة (JSON-LD)، وسوم Open Graph، وروابط قانونية (canonical).
- **إتاحة (Accessibility):** توافق مع معايير WCAG AA — تباين ألوان، تنقّل بلوحة المفاتيح، تسميات ARIA، ودعم `prefers-reduced-motion`.
- **الوضع الليلي** تلقائيًا حسب تفضيل النظام، مع إمكانية التبديل اليدوي.
- **أداء عالٍ:** أصول خفيفة، تحميل خطوط غير معطِّل، وتخزين مؤقت طويل الأمد.

## 🗂️ هيكل المشروع

```
.
├── index.html                 # الصفحة الرئيسية + المحول متعدد الفئات
├── converters/                # صفحة مخصّصة لكل فئة تحويل
│   ├── length.html  weight.html  area.html  temperature.html  volume.html
├── blog/                      # المدونة والمقالات
├── about.html  contact.html  privacy.html  404.html  offline.html
├── assets/
│   ├── css/style.css          # نظام التصميم الكامل
│   ├── js/converter.js        # محرّك التحويل
│   ├── js/main.js             # سلوك الواجهة (القائمة، الوضع الليلي، PWA)
│   └── img/                   # الأيقونات، الشعار، وصورة المشاركة
├── manifest.webmanifest       # بيان PWA
├── sw.js                      # عامل الخدمة (Service Worker)
├── sitemap.xml  robots.txt    # ملفات SEO
├── netlify.toml               # إعدادات النشر والأمان
└── build_*.py                 # مولّدات الصفحات الثابتة (للصيانة)
```

## 🛠️ التطوير المحلي

الموقع ثابت بالكامل (HTML/CSS/JS) بلا خطوة بناء. لتشغيله محليًا:

```bash
python3 -m http.server 8080
# ثم افتح http://localhost:8080
```

لإعادة توليد الصفحات بعد تعديل المحتوى في سكربتات البناء:

```bash
python3 build_pages.py && python3 build_content.py && python3 build_blog.py
```

## 🚀 النشر

يُنشر الموقع تلقائيًا على **Netlify** من الفرع `main`. الإعدادات معرّفة في `netlify.toml`
(مجلد النشر، ترويسات الأمان، والتحويلات). لا توجد أسرار في المستودع.
