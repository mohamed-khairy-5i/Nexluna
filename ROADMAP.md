# Nexluna — خطة التطوير والتوسيع المستقبلية
_مُعدّة بصفة خبير تطوير — مرتّبة حسب الأثر مقابل الجهد (Impact / Effort)_

الوضع الحالي: 14 محوّلًا، مدوّنة (4 مقالات)، بحث ذكي حتمي، PWA، تصميم فاخر، أداء FCP ~260ms، مساهم واحد على `main`.

---

## المبادئ الحاكمة
1. **البقاء ثابتًا (static-first):** لا خادم، لا تكلفة تشغيل — كل شيء يُبنى بـ Python إلى HTML/CSS/JS.
2. **صحّة دائمة، بلا هلوسة:** أي ميزة "ذكية" حتمية تمرّ عبر `window.NexConvert`.
3. **الأداء غير قابل للتفاوض:** لا نضيف ميزة تكسر المسار الحرج.
4. **SEO عربي أولًا:** كل توسّع = صفحة قابلة للفهرسة بكلمات مفتاحية حقيقية.
5. **RTL + إتاحة (a11y) في كل مكوّن جديد.**
